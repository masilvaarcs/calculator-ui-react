# Calculator UI - React

Interface web de calculadora construída com **React 18**, integrada a um backend via API REST.

## Funcionalidades

- Operações: soma, subtração, multiplicação e divisão
- Validação de entrada (apenas números válidos)
- Tratamento de divisão por zero
- Integração com backend configurável via variável de ambiente

## Stack

- React 18
- JavaScript (ES6+)
- Fetch API

## Configuração

```bash
REACT_APP_BACKEND_URL=http://localhost:3001
npm install
npm start
```

## Testes

```bash
npm test
```
